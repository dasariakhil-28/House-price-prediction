# House Price Prediction using Machine Learning

## Project Overview
This project predicts house prices using machine learning algorithms. 
The dataset contains various features such as location, size, number of bedrooms, and other housing attributes.

## Technologies Used
Python
Scikit-learn
Pandas
NumPy
Matplotlib
XGBoost

## Dataset
The dataset used for this project is the House Prices – Advanced Regression Techniques dataset.

## How to Run the Project

1. Clone the repository

git clone https://github.com/dasariakhil-28/House-price-prediction.git

2. Navigate to the project directory

cd House-price-prediction

3. Install required libraries

pip install -r requirements.txt

4. Run the model

python model.py

## Model Used

Linear Regression  
Decision Tree  
Random Forest  
XGBoost

## Evaluation Metric
pip install -r requirements.txt
python main.py
RMSE (Root Mean Square Error)

## Output

The model predicts house prices based on input features and evaluates performance using RMSE.
